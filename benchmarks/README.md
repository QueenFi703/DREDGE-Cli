# Benchmarks

This directory contains benchmark scripts, results, and related files for the DREDGE project.

## Contents

### Scripts
- **benchmark_demo.py** - Demonstration benchmark
- **quasimoto_benchmark.py** - Quasimoto benchmark
- **quasimoto_6d_benchmark.py** - 6D benchmark
- **quasimoto_extended_benchmark.py** - Extended benchmark
- **quasimoto_interference_benchmark.py** - Interference benchmark
- **migrate_quasimoto.sh** - Migration script for Quasimoto files

### Results
- **quasimoto_4d_convergence.png** - 4D convergence results
- **quasimoto_6d_convergence.png** - 6D convergence results

- **quasimoto_comparison.png** - Comparison results
- **quasimoto_convergence.png** - Convergence results
- **quasimoto_interference_comparison.png** - Interference comparison
- **quasimoto_interference_convergence.png** - Interference convergence
